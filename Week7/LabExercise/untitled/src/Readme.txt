Create a class called Person where, id, firstName,
lastName, and age attributes are declared.
  Create four constructors
 ◦ In first, set all attributes;
 ◦ In the second, assign only first and last names,
automatically increment id and age will be set to zero;
 ◦ In the third, only set id, and set rest to default values;
 ◦ Int the last only increment id, set the rest to default
values

Aşağıdaki gibi bir Person sınıfı oluştur:

id, firstName, lastName ve age (yaş) özelliklerini tanımlayın.

Dört tane kurucu (constructor) oluşturun:

    İlkinde, tüm özellikleri ayarlayın.
    İkincisinde, yalnızca firstName ve lastName özelliklerini atayın; id otomatik olarak artırılsın ve age sıfır olarak ayarlansın.
    Üçüncüsünde, yalnızca id ayarlansın, geri kalan özellikler varsayılan değerlere ayarlansın.
    Sonuncusunda, yalnızca id artırılsın ve geri kalan özellikler varsayılan değerlere ayarlansın.
