QUESTION.3 [24pt]: Write necassary lines of code to perform followings
a. Create a class called EncapsulatedNFT, which applies encapsulation (data hiding) to given NFT class.
public class NFT {
 public double balance; // balance cannot be less than 0 or greater than 10000
public String owner;
}
b. EncapsulatedNFT class should throw an exception (with an error message) if any illegal argument (balance) is set.
c. Inside the MainClass, create an instance/object from EncapsulatedNFT, set attributes for it, handle any exception, and print
the exception message on the screen.


Soru.3
a. NFT classina encapsulation (kapsülleme, veri gizleme) prensibini uygulayan EncapsulatedNFT adlı bir class oluşturun.
b. Illegal bir arguman ( balance) girilirse, EncapsulatedNFT sınıfı bir exception (bir hata mesajıyla) firlatmalidir.
c. MainClass’ta EncapsulatedNFT sinifindan bir nesne (instance) oluşturun ve degiskenleri atayın, herhangi bir exception
durumuyla başa çıkın (handle edin) ve istisna mesajını ekrana yazdırın.
